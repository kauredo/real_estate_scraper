## 🔄 Syncing

These files should be updated periodically from Notion:

1. Export pages from Notion as Markdown
2. Update corresponding files in this directory
3. Commit changes to git

**Frequency:** After major content updates or monthly.

## 🎯 Purpose

- **Version control** - Track changes to documentation
- **Backup** - Local copy of all Notion content
- **Offline access** - Read docs without internet
- **Git history** - See evolution of documentation

## ⚠️ Note

These are **snapshots** only. For the latest content and full Notion features (databases, embeds, etc.), always refer to the Notion workspace.
